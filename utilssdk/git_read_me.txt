To Push data on this repository follow the below steps:
1.Update the desired code in at location D:\WdUtilsSdk\utilssdk (where "utilssdk" is the package of SDK)
2.After changes copy all the data inside "utilssdk" package except "build" and "libs" folder.
3.Then Navigate to "D:\new sdk\wd-utils-sdk" 
4.Paste the copied data here and replace the files if already available
5.Now open "Git bash" treminal in the same path and Push data to the gitlab
6.Done !
