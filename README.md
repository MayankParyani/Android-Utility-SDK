# Download
Use Gradle:

```top level gradle
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

dependencies {
  implementation 'com.github.MayankParyani:Android-Utility-SDK:0.1'
}
```
